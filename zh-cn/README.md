<h1 align="center">Loon(气球)</h1>

> 企业级类Spring 基于typescript 和 express 的 Nodejs 框架

<h2 align="center">功能</h2>

* 依赖注入
* 服务类
* Result 路由和控制器
* 中间件
* 过滤器
* 模型
* 转化器
* 启动器

请参考 [文档](https://loon-project.github.io/#/zh-cn/) 获得更多信息。
